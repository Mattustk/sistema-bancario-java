# Sistema Bancário em Java

Projeto simples desenvolvido em Java para simular operações bancárias no terminal.

## Funcionalidades
- Depósito
- Saque
- Consulta de saldo
- Encerramento do programa

## Conceitos Utilizados
- Estruturas condicionais (if/else)
- Laço de repetição (while)
- Variáveis
- Entrada de dados com Scanner

## Objetivo
Projeto de estudo para praticar lógica de programação e fundamentos da linguagem Java.
